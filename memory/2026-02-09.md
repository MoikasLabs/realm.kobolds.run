### Technical Details

#### Building Properties
- `id`: Unique identifier
- `name`: Display name
- `type`: office/service/living/social/commerce
- `position`: {x, y, z} in 3D space
- `size`: {width, depth, height}
- `color`: Base building color
- `glowColor`: Glow light color
- `allowedRoles`: Who can use it
- `isOccupied`: Current occupancy status
- `occupants`: Array of agent IDs inside

#### Pathfinding
- Linear interpolation between waypoints
- Building entrance calculation with offset
- Distance utilities (XZ plane)
- Ready for agent movement in Phase 2
